# react_netflix

#### 1. 프로젝트 Description
- The movie DB API를 이용해서 만든 넷플릭스 웹 사이트입니다. 
API url요청을 위해 Axios 인스턴스를 별도로 만들어 메인화면 이미지 배너에 사용하였습니다.  
반응형으로 제작되었고 검색 창을 입력시 useDebounce를 생성하여 시간을 제어했습니다.
